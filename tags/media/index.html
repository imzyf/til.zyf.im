Mac 上录制和编辑 GIF

录制

LICEcap 小巧好用。

编辑

Mac 自带「预览」可以删 GIF 帧，但保存后循环属性丢了，只播放一次。改用 Photoshop：

  1. 打开 GIF，窗口 → 时间轴 编辑各帧
  2. 文件 → 导出 → 存储为 Web 所用格式
  3. 在「循环选项」里选 永久，再 存储

同一个面板里改「图像大小」可以顺手压缩。
