带中文注释的 Bash：

# 把当前目录下所有 Markdown 文件按修改时间排序
ls -lt *.md | head -10

# 替换文件中的全角逗号为半角（保留空格）
sed -i '' 's/，/, /g' *.md

Go 示例——通道传递中文字符串：

package main

import "fmt"

func main() {
    ch := make(chan string, 1)
    ch <- "通过通信共享内存，而不是通过共享内存通信"
    fmt.Println(<-ch)
}
