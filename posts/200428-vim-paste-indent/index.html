vim 终端粘贴自动缩进错乱用 :set paste

2020-04-28 #vim

从浏览器 / 别的终端 app 粘贴一段已缩进的代码到终端 vim，每行缩进会层层叠加变成阶梯——因为终端 vim 收不到”这是粘贴”的信号，把每个字符当成键盘输入处理，于是每按一次回车就按上一行结尾的缩进起步。

粘贴前后切回来：

:set paste
" 粘贴...
:set nopaste

或者一行 toggle：:set invpaste。

GUI vim（gvim / MacVim）不需要——它能直接区分粘贴事件和键盘输入。Neovim 在多数现代终端也能自动识别 bracketed paste，可以不管。
