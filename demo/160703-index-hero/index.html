2016-07-03 #demo

The newest note shows up here in full — a 60ch slot above the masonry grid. Everything else is a preview card.

Astro returns collection entries in disk order, not date order. Sort first, then split:

const notes = (await getCollection('notes')).sort(
  (a, b) => b.data.date.valueOf() - a.data.date.valueOf(),
);
const [hero, ...rest] = notes;

That’s the whole trick.
