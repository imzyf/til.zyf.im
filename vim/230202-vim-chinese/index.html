Vim 中文资源

2023-02-02 #vim

Vim 自带英文教程和帮助文档，中文化分两条路：跑自带教程时切语言，装社区维护的帮助文档包。

vimtutor 是 Vim 自带的 30 分钟交互式入门，环境变量切到中文 locale 就能直接读：

LANG=zh_CN.UTF-8 vimtutor

vimcdoc 是 yianwillis/vimcdoc 维护的中文帮助包，放进 pack/start 即随 Vim 加载，:help 直接出中文：

mkdir -p ~/.vim/pack/my/start
cd ~/.vim/pack/my/start
git clone https://github.com/yianwillis/vimcdoc.git
